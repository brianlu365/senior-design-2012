OpenCV data type:
  CvPoint - point in an image (x,y)
  CvSize - size of an image (width, height)
  CvRect - portion of an image (x, y, width, height)
  CvScaler - RBGA value for a pixel (val[4])
  cvMat* - matrix (rows, cols, type) for a 2 dimensional matrix
